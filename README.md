# Monitoring Stack — Prometheus & Grafana 📊

## Overview
Complete monitoring stack with Prometheus, Grafana and Node Exporter
deployed using Docker Compose.

## Architecture
Node Exporter → Prometheus → Grafana Dashboard

## Tech Stack
- Prometheus — Metrics collection
- Grafana — Visualization & Dashboards
- Node Exporter — System metrics
- Docker Compose — Orchestration

## How to Run
git clone https://github.com/AsmaSlimanii/monitoring-stack-prometheus-grafana.git
cd monitoring-stack-prometheus-grafana
docker-compose up -d

## Access
Prometheus : http://localhost:9090
Grafana    : http://localhost:3000
             admin / admin123
Node Exporter : http://localhost:9100
